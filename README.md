# NFC Story Project - 08 最终部署版

这是从 `07-solo优化版` 复制出来的最终部署目录，`07-solo优化版` 未修改。

## 本版内容

- 页面入口：`index.html`
- 故事数据：`js/data.js`
- 页面逻辑：`js/app.js`
- 样式文件：`css/style.css`
- 本地 GSAP：`vendor/gsap/gsap.min.js`
- 背景音乐：`assets/music/bgm.mp3`

## 本次调整

- 已将旧的测试图片全部替换为 `图片` 文件夹中的 15 张正式图片。
- 已删除复制时带入的旧 `assets/images/test*.jpg` 随机图。
- 主场景卡片改为更适合竖图的稳定比例，避免动画过程中尺寸跳动。
- 详情页支持按图片方向展示：竖图使用 3:4，偏横图使用 4:3。
- 详情页打开时隐藏音乐按钮，避免遮挡正文；返回主场景后自动恢复。
- 背景音乐已裁掉末尾提示音，当前时长约 34.04 秒，并在结尾做了淡出。
- 开场照片层结束后新增满屏爱心爆发转场，随后第一张照片单独稳定停留，再进入后续时间线。
- 资源版本号已更新，减少部署后浏览器缓存旧文件的概率。
