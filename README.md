# 旋律时间线故事集 - Solo 优化版

基于 Codex 升级版的优化版本，修复了关键问题并提升了性能和用户体验。

## 优化内容

### 1. 修复启动层隐藏问题（重要）
- **问题**：原版本点击"轻触开启故事"后，启动层可能仍然拦截点击事件
- **修复**：
  - 立即设置 `pointer-events: none` 禁用点击
  - 动画完成后设置 `display: none` 和 `visibility: hidden`
  - 添加 `aria-hidden="true"` 提升可访问性
  - 防止重复点击的标记保护

### 2. 添加数据检查和错误处理
- 验证 `storyData` 是否存在、是否为数组、是否为空
- 验证每个节点的必要字段（title, image）
- 数据错误时显示友好的错误提示
- 添加加载界面，提升等待体验

### 3. 优化 DOM 缓存和性能
- 集中缓存常用 DOM 元素，避免重复查询
- 使用 `getElementById` 替代 `querySelectorAll` 获取缓存元素
- 减少动画帧中的 DOM 操作

### 4. 添加音频错误处理
- 音频初始化使用 try-catch 包裹
- 添加 `error` 事件监听
- 播放失败时优雅降级，不影响其他功能
- 音乐按钮在未初始化时可点击触发初始化

### 5. 支持减少动画偏好（prefers-reduced-motion）
- 检测用户系统设置中的"减少动画"偏好
- 减少动画模式下：
  - 跳过 3D 开场动画
  - 减少粒子数量（16个 → 8个）
  - 移除爱心和流光效果
  - 禁用飞行动画，改为静态圆形布局
  - 直接显示所有节点，不隐藏

### 6. 图片预加载优化
- 使用 Promise 封装预加载逻辑
- 等待所有图片加载完成后再显示启动界面
- 添加 10 秒超时保护，防止加载卡住
- 单张图片加载失败不影响整体流程

### 7. 内存泄漏防护
- 页面卸载时清理所有定时器
- 停止动画帧
- 停止音乐播放
- 释放音频对象引用

### 8. 其他优化
- 使用 `pointerdown` 替代 `click` 获得更快响应
- 图片添加 `loading="eager"` 优先加载
- 添加触摸设备检测（为后续优化预留）
- 代码结构优化，使用 CONFIG 对象集中管理常量
